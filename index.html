Programming Languages and Tools

I have good experience with Python-based libraries for data analysis, including multi-dimensional data visualization, map construction and geological modelling. Projects

Python-based projects and solutions

Achilles.BR

A Python-based application I've developed together with LOOP as part of my PhD research. This application is capable of reading commonly used well log data from LAS files, interpolate data, provide statistical insights, clusterize samples and export the outputs. This app is not open for public use, but its main window can be visualized here.
